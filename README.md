## Setup

- CLONE this repo onto your computer
- Go to your terminal/command line and navigate to the DonutCat2 folder.
- Paste the appropriate commands into your terminal

<h3>Setup (first time only)</h3>


    make setup
    



<h3>To  play!</h3>


    make run
